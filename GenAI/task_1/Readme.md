# LangChain Prompt Templates – Mini Prompt Engine

## Overview

This project demonstrates how to build dynamic and reusable prompt systems using LangChain PromptTemplate and ChatPromptTemplate.

The notebook covers:

* PromptTemplate basics
* Multi-input prompt systems
* Prompt variations
* Chat-based prompts
* Input validation
* Dynamic prompt generation
* Template reusability

## Technologies Used

* Python
* LangChain
* Jupyter Notebook

## Features

* Reusable prompt templates
* Dynamic user input handling
* Prompt validation system
* Multiple prompt styles
* ChatPromptTemplate implementation

## Project Structure

* Task 1 → Hardcoded Prompt Replacement
* Task 2 → Multi-Input Prompt System
* Task 3 → Prompt Variations Engine
* Task 4 → ChatPromptTemplate System
* Task 5 → Input Validation
* Task 6 → Prompt Generator App
* Task 7 → Template Reusability Test

```bash
pip install langchain
```

Open the notebook and execute all cells.
