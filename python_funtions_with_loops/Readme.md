# Python Functions with Loops Assignment

## Overview

This repository contains solutions for the Python Functions and Loops Assignment completed as part of the Data Science Internship Program.

The assignment focuses on solving real-world programming problems using:

* Python Functions
* Loops
* Conditional Statements
* Lists
* Dictionaries
* String Handling

---

# Problems Solved

## Problem 1: Mobile Recharge Validation System

* Recharge amount validation
* Plan verification
* Retry option using while loop

## Problem 2: Inventory Reorder Alert System

* Product stock checking
* Reorder alert generation

## Problem 3: Student Result Processing System

* Average marks calculation
* Pass/Fail evaluation

## Problem 4: Cab Fare Estimator

* Distance-based fare calculation
* Peak hour extra charges
* Retry functionality

## Problem 5: Employee Attendance Eligibility Checker

* Attendance percentage calculation
* Eligibility verification

## Problem 6: Password Strength Checker

* Password validation
* Digit and special character checking
