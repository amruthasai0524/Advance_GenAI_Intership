# NLP Pipeline Blog Project

## Overview

This project explains the complete NLP (Natural Language Processing) pipeline from raw text preprocessing to feature extraction and vectorization techniques.

The project includes:

* NLP preprocessing concepts
* Text cleaning challenges
* Feature engineering techniques
* Comparison of vectorization methods
* End-to-end NLP workflow

---

## Topics Covered

### Text Preprocessing

* Lowercasing
* Stopword Removal
* Tokenization
* Stemming
* Lemmatization

### Text Cleaning

* Emojis
* URLs
* Special Characters
* Noisy Text

### Feature Engineering

* Bag of Words (BoW)
* TF-IDF
* Word2Vec
* Average Word2Vec
