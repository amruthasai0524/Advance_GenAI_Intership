# AI Chatbot using Transformers

## Project Overview

This project is a conversational AI chatbot developed using Hugging Face Transformers and Microsoft DialoGPT model.

The chatbot accepts user input and dynamically generates meaningful responses using a pre-trained transformer-based language model.

## Technologies Used

* Python
* Hugging Face Transformers
* PyTorch
* DialoGPT

## Features

* Interactive chatbot
* Dynamic AI-generated responses
* Continuous conversation handling
* Exit condition support
* Console-based interaction

## Model Used

* microsoft/DialoGPT-medium

## How to Run

### Install Dependencies

```bash
pip install transformers torch
```

### Run the Notebook

Open the Jupyter Notebook and execute all cells.

## Expected Output

The chatbot communicates naturally with users and generates intelligent responses.
