# Fine-Tuning DistilBERT for POS Tagging and Chunking

## Project Overview

This project demonstrates fine-tuning DistilBERT for token classification tasks:

* Part-of-Speech (POS) Tagging
* Chunking (Phrase Detection)

The project uses Hugging Face Transformers and PyTorch.

---

## Technologies Used

* Python
* Hugging Face Transformers
* PyTorch
* Datasets
* Seqeval
* Jupyter Notebook

---

## Datasets Used

### POS Tagging

### Chunking

* CoNLL-2003
