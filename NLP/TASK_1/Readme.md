# NLP Preprocessing Engine

## Overview

This project implements a robust NLP preprocessing engine for cleaning real-world noisy text data.

The preprocessing pipeline includes:

* Lowercasing
* URL removal
* Email removal
* Number removal
* Repeated character normalization
* Extra whitespace removal
* Tokenization
* Short token filtering
* Lemmatization

## Features

* Handles noisy social media text
* Removes emojis and unwanted symbols
* Performs token analytics
* Generates frequency analysis
* Includes modular reusable functions
* Handles edge cases and errors

## Technologies Used

* Python
* NLTK
* Regular Expressions
* Collections Counter
