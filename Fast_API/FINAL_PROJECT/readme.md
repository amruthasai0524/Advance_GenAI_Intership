# FastAPI Library Book Management System

## Project Overview

This project is a complete backend application built using FastAPI as part of the FastAPI Internship Final Project.

## Features

* REST APIs
* CRUD operations
* Pydantic validation
* Helper functions
* Multi-step workflow
* Search functionality
* Sorting APIs
* Pagination APIs
* Swagger documentation

## Technologies Used

* Python
* FastAPI
* Pydantic
* Uvicorn

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run server:

uvicorn main:app --reload

## Swagger UI

http://127.0.0.1:8000/docs
